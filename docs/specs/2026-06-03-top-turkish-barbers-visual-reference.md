## Purpose

This document provides the primary visual direction for the UI refinement phase.
The website currently looks like the current state image attached ( ![current-state](./current-state.png) ) inside this folder
The attached reference image ( ![visual-reference](./visual-reference.png) ), also inside this folder, should be treated as the primary source of truth for:

- layout proportions
- section sizing
- visual hierarchy
- composition
- spacing rhythm
- premium visual tone

The written specs remain the source of truth for:

- behaviour
- content
- implementation constraints
- acceptance criteria

## Important instruction

The reference image is NOT a pixel-perfect design target.

Implementation should capture:

- overall feel
- visual hierarchy
- section density
- premium composition
- luxury aesthetic

while remaining:

- responsive
- maintainable
- implementation-friendly

## Desired visual characteristics

### Premium

The website should feel:

- high-end
- luxurious
- established
- trustworthy

Avoid:

- template appearance
- generic SaaS styling
- excessive decoration

### Compact

The website should feel:

- curated
- intentional
- concise

Avoid:

- giant stacked sections
- excessive vertical whitespace
- long-scroll behaviour

### Elegant

Motion and styling should feel:

- restrained
- smooth
- deliberate

Avoid:

- flashy animation
- gimmicky effects
- excessive movement

## Section reference priorities

### Highest priority

1. Services layout
2. Gallery composition
3. Overall spacing rhythm
4. Section proportions

### Medium priority

1. Typography hierarchy
2. Card styling
3. Decorative details

### Lower priority

1. Exact imagery
2. Exact icon placement
3. Exact ornament styling

## Services reference

Target characteristics:

- card-based layout
- integrated pricing
- premium icons
- compact composition

Avoid:

- menu appearance
- PDF appearance
- bullet-list appearance

## Gallery reference

Target characteristics:

- portrait imagery
- framed composition
- compact presentation
- curated feel

Avoid:

- giant image blocks
- full-width placeholder areas
- excessive section height

## Location section reference

Target characteristics:

- balanced 2-column layout
- strong information grouping
- compact presentation

Avoid:

- long stacked content
- weak hierarchy
